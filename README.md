随着人工智能的飞速发展，医疗问答系统的相关技术不断进步并普及，医患用户的对话需求也日益多元化。传统人工定义的医疗问答任务往往无法完全覆盖医患的所有需求，无法提供精准的病情诊断和优秀的治疗手段。现有的方法面临以下三个挑战：首先，目前的问答任务基准数据集仅包含单一问题与若干答案，而缺少对问题个性化信息的相关文本信息；其次，现有方法往往忽略了医疗领域内的领域信息对问答任务训练带来的提升效果，从而导致问答任务的性能十分有限；最后，由于训练数据量有限，现有方法在预训练模型仍存在局限性，如何将开放域数据有效利用在问答任务上是至关重要。为了应对这些挑战，本文面向医疗领域开放域问答任务方法进行研究，主要研究内容如下：
1）提出基于预训练模型的个性化信息分类任务的方法。现有问答任务方法通常使用通用的模型作为基础来完成预训练任务，从而导致进行医疗领域的问答任务性能有限。为了解决上述问题，首先使用医疗领域预训练语言模型作为文本数据的个性化信息提取器，在训练过程中将医患用户的对话文本内容作为输入并进行编码，使用本文收集的高质量医疗问答数据集进行预训练，从而使模型捕获到浅层的问答任务语义关系；其次，再进一步学习来对医疗问答任务进行个性化信息的分类，通过训练让预训练模型能够更注重医疗个性化信息，完成医疗个性化信息的分类任务；最后，使用完成微调的模型对相关数据集完成信息分类，以便下游任务使用。该方法在三个数据集CMDD、Huatuo-26M和webMedQA上都取得了超过所有最优基线的实验结果，在Huatuo-26M与webMedQA数据集中较BERT基线模型有可观的提升。
2）提出一种基于二次文本检索的问答任务方法。现有的文本检索技术仅根据定义的规则来进行检索，很难满足发现医患用户更加细致的文本检索的需求，利用多轮的文本检索以及先进的文本检索技术来完成检索任务是迫切需要的。为了解决上述问题，首先，本文收集了一个信息完善的医疗数据集，其中包含医患的所有对话内容以及医患各自的详细信息，并通过收集、清洗、切割等自动化处理过程获得大量的高质量对话数据。其次，通过初级的文本检索技术完成相关医患对话记录的匹配筛选出可供使用的对话记录。同时设计并构建了一个二次检索流程，通过初级检索与二次DPR检索完成对数据的筛选；最后，通过搭建问答任务基线实验验证医疗问答数据的有效性。该方法发在数据集Ask39Med上取得了超过所有最优基线的实验结果。
3）提出一种基于检索增强生成的医疗问答任务方法。现有的问答任务方法仅包含有少量医疗数据，往往无法匹配医患用户的医疗问答现实需求。为了解决上述问题，采用以下方法。首先，利用二次检索进行有监督预训练，以学习开放域问答任务的语义信息；其次，为了充分利用个性化信息，在进行训练之前，本文引入了个性化信息分类算法，以便推测每个医疗问题的个性化分类结果，本文通过在二次检索以及开放域问答训练的过程中均融入个性化信息。这种改进能够进一步有利于避免回答非相关科室医疗信息的不稳定影响，使得答案生成结果更加稳定和可靠。最后，使用经过检索后得到若干高质量答案作为外挂知识库，用于进行高质量开放域任务问答训练，经过预训练模型微调以完成对任务医疗开放域问答任务的推理。该方法在三个数据集Ask39Med和cMedQA2.0上都取得了超过所有最优基线的实验结果，比BART基线模型最多均有8%的提升。
4）设计并实现了一个基于检索增强生成的医疗问答集成平台。现存的预训练模型已经实现在医疗文本数据中完成医疗问答的任务，但由于对比实验的复杂性很高，并且缺乏更直观的评价指标，所以在基线模型的生成诊断结果方面仍有较大研究空间。基于此，设计并实现了一个集成二次检索、检索增强生成、交互式的可视化模型平台。它集成了多种先进的预训练模型以及基线模型，用户可以使用医疗科室个性化信息分类、检索相关医疗案例以及生成式医疗问题诊断与治疗建议在内的三大模块功能。
综上所述，经实验证明提出的方法通过推测个性化分类信息、二次检索以及开放域问答任务训练分别取得了一定的性能提升，为检索增强生成与开放域医疗问答提供了参考方向。这一整合方法有助于提高模型的性能，在开放域问答任务数据集cMedQA2.0上取得显著效果。相关数据集与相关算法开源于Github 项目https://github.com/lihuan777/MedicalQA。所有源代码、原始数据、中间生成结果及最终结果文件均存放于通过网盘分享的文件：https://pan.baidu.com/s/1PnA3dLZ5eg93VElXJ5VTjQ 提取码: a6t8 。
